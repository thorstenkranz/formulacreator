About {{title}}

Who?

This project was started by Thorsten Kranz.

Why?

The initial intend was purely educational - just to get started with Google App Engine and see what's possible. After a couple of hours of work, an initial version was up and running and turned out to be quite useful.

How?

This project is hosted at Google App Engine. It's server-part is written in Python, the formula rendering is done with Matplotlib. For the client-side, jQuery and jQuery UI are used to spice up the interface.
